# SchuleIntern - Backup Restore


## How To Use

- Download Repro
- Move "www/restore" to your Webserver
- open YOUR_URL/restore





## Development


### Project init
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run release
```

### PHP Modification

at ./restore.php
NOT ./www/restore/restore.php !!!
